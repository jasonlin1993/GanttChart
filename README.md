## 專案展示
「GanttChart」線上繪製屬於你的工程甘特圖，可輕鬆選擇月份以及不同任務顏色，並且輸出為 PDF 檔案，將你的工程專案視覺化，簡單的在一個頁面中呈現。
### 功能亮點
- 顏色自定義：為不同的任務選擇個性化的顏色標記
- 即時編輯：簡單的新增和刪除任務，方便調整專案計畫
- PDF導出：一鍵將你的甘特圖轉換為PDF格式，便於分發和列印
- 實時同步：使用 Firebase Firestore 資料庫，保證你的數據實時同步和備份
- 會員登入系統：使用 Firebase Authentication 作為會員登入登出功能
